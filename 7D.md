NIP-7D
======

线程
-------

`草案` `可选`

线程是一个 `kind 11` 事件。线程应该包含一个 `title`。

```json
{
  "kind": 11,
  "content": "早上好",
  "tags": [
    ["title", "GM"]
  ]
}
```

对 `kind 11` 的回复必须使用 [NIP-22](./22.md) `kind 1111` 评论。回复应该
始终是对根 `kind 11` 的回复，以避免任意嵌套的回复层次结构。

```json
{
  "kind": 1111,
  "content": "很棒",
  "tags": [
    ["K", "11"],
    ["E", <事件ID>, <中继URL>, <公钥>]
  ]
}
```
